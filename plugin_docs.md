# Plugin Documentation

This file provides a summary of the installed Neovim plugins and their purpose.

## Core Plugins

*   **goolord/alpha-nvim**: A greeter for Neovim. It is used to display the dashboard you see on startup.
*   **catppuccin/nvim**: A soothing pastel theme for Neovim.
*   **lazy.nvim**: The plugin manager itself.

## File Navigation and Searching

*   **nvim-telescope/telescope.nvim**: A highly extendable fuzzy finder over lists. Used for finding files, buffers, help tags, and more.
*   **nvim-lua/plenary.nvim**: A dependency for many plugins, providing useful Lua functions.
*   **nvim-telescope/telescope-project.nvim**: An extension for Telescope to easily switch between projects.
*   **nvim-tree/nvim-tree.lua**: A file explorer for Neovim.
*   **ibhagwan/fzf-lua**: Another fuzzy finder, based on fzf.
*   **mikavilpas/yazi.nvim**: A Neovim plugin for integrating with the Yazi terminal file manager.

## Coding Assistance

*   **williamboman/mason.nvim**: Manages LSP servers, DAP servers, linters, and formatters.
*   **williamboman/mason-lspconfig.nvim**: A bridge between mason.nvim and nvim-lspconfig.
*   **neovim/nvim-lspconfig**: Configurations for the built-in LSP client.
*   **hrsh7th/nvim-cmp**: A completion engine for Neovim.
*   **hrsh7th/cmp-nvim-lsp**: A source for nvim-cmp for LSP completions.
*   **hrsh7th/cmp-buffer**: A source for nvim-cmp for buffer completions.
*   **hrsh7th/cmp-path**: A source for nvim-cmp for path completions.
*   **hrsh7th/cmp-cmdline**: A source for nvim-cmp for command-line completions.
*   **saadparwaiz1/cmp_luasnip**: A source for nvim-cmp for Luasnip snippets.
*   **numToStr/Comment.nvim**: A plugin for easily commenting out lines of code.
*   **Wansmer/treesj**: A plugin for joining and splitting code blocks based on tree-sitter.
*   **RRethy/vim-illuminate**: Highlights other uses of the word under the cursor.

## Git Integration

*   **lewis6991/gitsigns.nvim**: A plugin to show git signs in the sign column.

## UI and UX

*   **nvim-tree/nvim-web-devicons**: Adds file-type icons to various plugins.
*   **nvim-mini/mini.animate**: Provides animations for Neovim.
*   **folke/noice.nvim**: Replaces the Neovim UI for messages, cmdline and popups.
*   **MunifTanjim/nui.nvim**: A UI component library, a dependency for noice.nvim.
*   **rcarriga/nvim-notify**: A notification manager for Neovim.
*   **folke/flash.nvim**: Enhances search and navigation within a buffer.
*   **nvim-lualine/lualine.nvim**: A statusline for Neovim.
*   **sphamba/smear-cursor.nvim**: A cursor effect plugin.
*   **karb94/neoscroll.nvim**: A plugin for smooth scrolling.
*   **y3owk1n/undo-glow.nvim**: A plugin that highlights undo/redo changes.
*   **NvChad/nvim-colorizer.lua**: A plugin to highlight color codes.
*   **folke/twilight.nvim**: Dims inactive portions of the code you're editing to improve focus.
*   **rachartier/tiny-devicons-auto-colors.nvim**: Automatically updates nvim-web-devicons colors based on your current colorscheme.
*   **SmiteshP/nvim-navic**: A simple statusline/winbar component that uses LSP to show your current code context (breadcrumbs).
*   **akinsho/bufferline.nvim**: A snazzy buffer line built using Lua, making it easier to manage and switch between open files.

## Debugging and Execution

*   **mfussenegger/nvim-dap**: A Debug Adapter Protocol client implementation for Neovim.
*   **mfussenegger/nvim-dap-python**: An extension for nvim-dap for debugging Python.
*   **jpalardy/vim-slime**: A plugin for sending text from Vim to a terminal.

## Miscellaneous

*   **christoomey/vim-tmux-navigator**: Allows seamless navigation between Vim and Tmux panes.
*   **uhs-robert/sshfs.nvim**: Mounts remote filesystems via SSHFS, allowing local Neovim to edit remote files.
